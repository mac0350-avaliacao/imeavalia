## Propósito

Repositório para o desenvolvimento de uma página web que permita realizar a avaliação discente semestral dos alunos do IME-USP.

O diagrama UML do projeto pode ser encontrado em .

## Tecnologias

* Backend: Kotlin
* Frontend: Vue3
